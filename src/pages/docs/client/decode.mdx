# Decodes

Decodes an encoded URL.

Using default service.

```js
const encodedURL = await chemical.encode("https://example.com")
await chemical.decode(encodedURL)
```

Decode with another service.

```js
const encodedURL = await chemical.encode("https://example.com")
await chemical.decode(encodedURL, {
    service: "uv"
})
```
