# Get Store

Gets a stored value used by Chamical.

Will return the default value if empty.

You can get: "transport", "wisp", "service", "autoHttps", "searchEngine"

```js
chemical.getStore("service")
```

You might want to use the [Select Component](/docs/components/select/) for select menus.
