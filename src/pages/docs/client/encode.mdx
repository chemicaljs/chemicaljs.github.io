# Encode

Generate an encoded URL.

Using default service, no auto https, and no search engine.

```js
await chemical.encode("https://example.com")
```

Advanced encoding with options.

```js
await chemical.encode("https://example.com", {
    service: "uv",
    autoHttps: true,
    searchEngine: "https://www.google.com/search?q=%s"
})
```
